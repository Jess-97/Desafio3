# 🌍 Country Explorer App

Aplicación móvil desarrollada con React Native y Expo para explorar países, ver información de sus capitales y consultar el clima actual.

## 📱 Pantallas
- SplashScreen
- LoginScreen
- RegisterScreen
- HomeScreen
- DetailScreen

## 🌎 APIs utilizadas
- Rest Countries API
- OpenWeatherMap API

## 🛠️ Tecnologías
- React Native
- Expo
- React Navigation
- React Native Maps
